GUIDE TO PLAY XYLOPHONE

Use your computer keyboard or click the pan-flute to play the virtual xylophone. You can use the following shortcuts:

  • C3 : use q
  • D3 : use w
  • E3 : use e
  • F3 : use r
  • G3 : use t
  • A4 : use y
  • B4 : use u
  • C4 : use a
  • D4 : use s
  • E4 : use d
  • F4 : use f
  • G4 : use g
  • A5 : use h
