GUIDE TO PLAY GUITAR

Use your computer keyboard or click the guitar to play the virtual guitar. You can use the following shortcuts:

  • 1st string : use e
  • 2nd string : use b
  • 3rd string : use g
  • 4th string : use d
  • 5th string : use a
  • 6th string : use s
